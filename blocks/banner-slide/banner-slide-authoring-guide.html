Banner Slide — Authoring Guide

Block name: banner-slide
Creates a full-width hero banner carousel with buttons, images, and custom colors.

1. Block Structure

Each banner-slide row has 4 columns:

#ColumnWhat to Put HereRequired?
1ContentTitle, description, and up to 2 buttonsYes
2FeaturesIcon + text highlightsNo
3BackgroundBackground image (1st = desktop, 2nd = mobile)No
4ForegroundProduct / character image on top of backgroundNo
Multiple slides: Add multiple banner-slide rows in the same section to create a carousel.

2. Content Column

  • Title: Format text as a Heading (H1–H6)
  • Description: Type as a normal paragraph
  • Short description: Create a bulleted list — appears below the buttons

3. How to Create Buttons

Add up to 2 buttons at the bottom of the content column:

Button TypeWhat It DoesHow to Author
Link Button Opens a URL Type text → Underline it → Add a hyperlink
Form Open Button Opens the loan form Type text → Underline it → Make it Superscript

Link Button

  1. Type the button text, e.g., Apply Now
  2. Select it and Underline (Ctrl + U)
  3. Add a Hyperlink (Ctrl + K) with the destination URL

Form Open Button

  1. Type the button text, e.g., Apply Now
  2. Select it and Underline (Ctrl + U)
  3. Apply Superscript (Ctrl + Shift + =)

Order does not matter — underline first or link/superscript first, both work.

Two Buttons

First button = Primary (filled). Second button = Secondary (outline).

Important: Only text at the bottom of the content column is checked for buttons. If text is not underlined + linked or underlined + superscript, it stays as description.

4. Features Column

Insert an icon image, then type the label on the next line. Repeat for each feature.

6. Custom Colors

Add color keywords in the block class field using round braces with a 6-digit hex code (no #).

KeywordWhat It ChangesExample
bgSlide backgroundbg(FEEBFF)
eyebrowTitle text coloreyebrow(FFFFFF)
descriptionDescription text colordescription(333333)
cta1Primary button textcta1(FFFFFF)
ctabg1Primary button backgroundctabg1(E03E2D)
cta2Secondary button textcta2(333333)
ctabg2Secondary button backgroundctabg2(FFFFFF)
Example:
banner-slide (bg(FEEBFF), eyebrow(FFFFFF), ctabg1(E03E2D))
→ Pink background, white title, red primary button.
Use 6-digit hex codes without # — e.g., bg(FF0000), not bg(#FF0000).

7. Special Classes

ClassEffect

green-textBold text in description becomes green
auto-rotateSlides rotate automatically every 7 seconds (add to section)
