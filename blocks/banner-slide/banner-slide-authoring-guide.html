Banner Slide — Authoring Guide

Block name: banner-slide
What it does: Creates a full-width hero banner carousel on the page. Supports multiple slides, call-to-action buttons, feature highlights, background images, foreground images, video, and custom colors.

1. Block Structure

Each banner-slide row in your spreadsheet has 4 columns (cells):

#ColumnWhat to Put HereRequired?
1ContentTitle (heading), description text, and up to 2 CTA buttonsYes
2FeaturesSmall icons with text labels (e.g., "Quick Approval", "Minimal Docs")No
3BackgroundBackground image — add 2 images for separate desktop & mobile versionsNo
4ForegroundProduct image or character image that appears on top of the backgroundNo
Multiple slides: To create a carousel with multiple slides, add multiple banner-slide rows inside the same section. They are automatically combined.

2. Content Column

2.1 Title

Type your slide heading. Use any heading level (Heading 1 through Heading 6) in the Word document or Google Doc.

Example: Type "Get a Loan Against Car in Just a Few Easy Steps" and format it as Heading 2.

2.2 Description

Type your description as a normal paragraph below the title.

Example: "Quick Approval | Minimal Documentation | Flexible Tenure"

2.3 Short Description (Bullet List)

If you add a bulleted list, it will appear below the CTA buttons as a short description area.

Example:

  • Interest rates starting 10.99%
  • Check eligibility

3. How to Create CTA Buttons

You can add up to 2 buttons at the bottom of the content column. There are two types:

Button TypeWhat It DoesHow to Author It
Link Button Clicking it opens a URL (navigates to another page) Type the button text → Underline it → Add a hyperlink to it
Form Open Button Clicking it opens the loan application form Type the button text → Underline it → Make it Superscript

3.1 Creating a Link Button (opens a URL)

  1. Type the button text on a new line, e.g., Apply Now
  2. Select the text and apply Underline (Ctrl + U)
  3. While still selected, add a Hyperlink (Ctrl + K) and paste the destination URL

The order does not matter — you can underline first then add the link, or add the link first then underline. Both work the same.

Result: The button will appear on the page as a styled button. When the user clicks it, they will be taken to the URL you entered.

3.2 Creating a Form Open Button (opens the loan form)

  1. Type the button text on a new line, e.g., Apply Now
  2. Select the text and apply Underline (Ctrl + U)
  3. While still selected, apply Superscript (Ctrl + Shift + =)

The order does not matter — you can underline first then superscript, or superscript first then underline. Both work the same.

Result: The button will appear on the page as a styled button. When the user clicks it, the loan application form will open.

3.3 Adding Two Buttons

If you add two buttons, the first one (top) will be styled as the Primary button (filled/solid) and the second one (bottom) will be styled as the Secondary button (outline).

Example with two buttons:
  1. Line 1: Apply Now — underlined + hyperlinked → becomes a Link button (Primary)
  2. Line 2: Check Eligibility — underlined + superscript → becomes a Form Open button (Secondary)
Important: Only text at the bottom of the content column is checked for buttons. Normal paragraphs above them stay as description text. If text is not underlined+linked or underlined+superscript, it will be treated as regular description text.

4. Features Column

Use this column to show small icon + text highlights (e.g., "Quick Approval", "Minimal Docs").

How to add a feature:
  1. Insert an icon image (e.g., a small SVG or PNG)
  2. On the next line, type the label text (e.g., "Quick Approval")
  3. Repeat for more features

You can also add text-only features (without an icon) — just type the text as a paragraph.

5. Background Images

Add images to the Background column for the slide's full-width background.

ImagePurpose
1st imageDesktop background (shown on large screens)
2nd image (optional)Mobile background (shown on screens smaller than 992px)

You can add images by:

  • Inserting an image directly
  • Adding a link to the image from the DAM
  • Pasting the image path as plain text
Performance note: The first slide's images load immediately for fast display. Other slides' images load only when the user scrolls to them.

6. Foreground Image

Add a product image, character, or illustration that appears on top of the background. This is typically placed on the right side of the slide.

  • Add 1 image for desktop, optionally a 2nd image for mobile
  • If you add a link to a video (not an image), it will show a "Watch Video" button instead

7. Custom Colors

You can customize the colors of different parts of the slide by adding special keywords in the block class field (the block name row in the spreadsheet).

7.1 Available Color Options

KeywordWhat It ChangesHow to Write ItVisual Result
bgSlide background colorbg(FEEBFF)Light pink background
eyebrowTitle / heading text coloreyebrow(FFFFFF)White title text
descriptionDescription text colordescription(333333)Dark grey description
cta1Primary button text colorcta1(FFFFFF)White text on primary button
ctabg1Primary button background & borderctabg1(E03E2D)Red primary button
cta2Secondary button text colorcta2(333333)Dark text on secondary button
ctabg2Secondary button background & borderctabg2(FFFFFF)White secondary button

7.2 How to Apply Colors

In the block name row of your spreadsheet, type the block name followed by the color keywords separated by commas:

Example:

banner-slide (bg(FEEBFF), eyebrow(FFFFFF), cta1(FFFFFF), ctabg1(E03E2D))

This gives you: pink background, white title, white text on a red primary button.

Rules for colors:
  • Use 6-character hex color codes — for example, FF0000 for red, FFFFFF for white, 000000 for black
  • Do NOT include the # symbol — write bg(FF0000), not bg(#FF0000)
  • The system automatically adds the # for you

7.3 Color Fallback

  • If you set an eyebrow color but not a description color, and the slide has no title, the eyebrow color will automatically apply to the description text as well.
  • Any other class you add (e.g., green-text) that is not a color keyword will be applied as a regular CSS class on the slide.

8. Auto-Rotate (Carousel)

To make slides rotate automatically, add the class auto-rotate to the section (not the individual block).

FeatureDetail
Rotation speedEvery 7 seconds
Pause on hoverYes — stops when the user hovers their mouse over the banner
Pause when tab hiddenYes — stops when the user switches to another browser tab
Pause when form openYes — stops when the loan form is open
Navigation dotsShown automatically when there are 2 or more slides
Mobile swipeUsers can swipe left/right on mobile to change slides
